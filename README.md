# English Around the World - Evaluation Form

A modern web-based evaluation form for teachers to assess curriculum materials on countries and cultures.

## Features

- 🧙 **Step-by-step wizard** - One question at a time, low cognitive load
- 🔄 **Multiple evaluations** - Teachers can evaluate multiple countries/levels in one session
- 📊 **Real-time data** - Evaluations saved to Supabase instantly
- 📥 **CSV export** - Download all evaluations as spreadsheet
- 📱 **Responsive design** - Works on desktop, tablet, mobile
- ⚡ **Fast** - Built with Next.js for optimal performance

## Quick Start

### 1. Setup Supabase Database

1. Create a new project at [supabase.com](https://supabase.com)
2. Create table `evaluations` with these columns:

```sql
CREATE TABLE evaluations (
  id BIGINT PRIMARY KEY GENERATED BY DEFAULT AS IDENTITY,
  docent_name TEXT NOT NULL,
  docent_email TEXT NOT NULL,
  land TEXT NOT NULL,
  jaar TEXT NOT NULL,
  niveaus TEXT[] NOT NULL,
  ratings JSONB NOT NULL,
  best_parts TEXT[],
  why_valuable TEXT,
  best_key_term TEXT,
  terms_to_remove TEXT,
  why_remove TEXT,
  replacement_suggestion TEXT,
  new_term TEXT,
  most_engaging_term TEXT,
  best_remembered_term TEXT,
  most_forgotten_term TEXT,
  level_fit INTEGER,
  too_easy_hard TEXT,
  student_difficulties TEXT,
  teacher_difficulties TEXT,
  one_change TEXT,
  created_at TIMESTAMP DEFAULT NOW()
);
```

3. Get your **Project URL** and **Anon Key** from Supabase settings

### 2. Setup Environment Variables

Create `.env.local`:

```
NEXT_PUBLIC_SUPABASE_URL=your_project_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_anon_key
```

### 3. Install and Run Locally

```bash
npm install
npm run dev
```

Visit `http://localhost:3000`

## Deployment to Vercel

1. Push code to GitHub:
```bash
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/your-username/english-around-evaluation
git push -u origin main
```

2. Deploy on [vercel.com](https://vercel.com)
   - Connect your GitHub repo
   - Add environment variables: `NEXT_PUBLIC_SUPABASE_URL` and `NEXT_PUBLIC_SUPABASE_ANON_KEY`
   - Set custom domain to `english-around-the-world-evaluation.vercel.app`
   - Deploy!

## How Teachers Use It

1. **Enter info** - Name & email (one-time)
2. **Evaluate countries** - Select country, year, levels, answer questions
3. **Repeat** - Add more countries if needed
4. **Review & export** - See summary, download CSV

## Tips for Efficiency

- Teachers can evaluate multiple years/countries in one sitting
- Auto-save to database means no data loss
- CSV includes all data for analysis in Excel/Google Sheets
- Optional fields for detailed feedback can be collapsed
